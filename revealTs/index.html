Attribute directives

Create a highlight directive that sets the background color of the host element to yellow.
							To create a directive, use the CLI command ng generate directive
							
								ng generate directive highlight
							
						
								The CLI generates the default src/app/highlight.directive.ts as follows:
								
								import {Directive} from '@angular/core';

								@Directive({
									standalone: true,
									selector: '[appHighlight]',
								})
								export class HighlightDirective {}
								
							
								The @Directive() decorator's configuration property specifies the directive's CSS attribute selector, [appHighlight].
								
								import {Directive, ElementRef} from '@angular/core';

								@Directive({
									standalone: true,
									selector: '[appHighlight]',
								})
								export class HighlightDirective {
									constructor(private el: ElementRef) {
										this.el.nativeElement.style.backgroundColor = 'yellow';
									}
								}

Build your first Angular app

  • Angular requires an active LTS or maintenance LTS version of Node and npm.
  • Install the Angular CLI which provides tooling for effective Angular development. 
    npm install -g @angular/cli
  • Creating a project start template 
    ng new [project_name] || ng n [project_name]
    								You can add flags:
								--routing, --style, --directory ./
  • Start your application 
    npm start || ng serve || ng s
